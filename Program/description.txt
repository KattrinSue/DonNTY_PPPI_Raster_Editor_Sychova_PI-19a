Растровый редактор, с реализацией не менее 20-ти программных средств доступных в Adobe Photoshop не ниже версии CS3.

	2)	создать документ
	3)	сохранить документ
Б) модуль редактирования проекта (ответственный: Сычева Екатерина)
	4)	вырезать
	5)	копировать
	6)	вставить
	7)	отменить действие
	8)	повторить действие
В) модуль редактирования изображения (ответственный: Сычева Екатерина)

	10)	Коррекция цветовой модели (Яркость, контрастность, кривые, экспозиция, цветовой баланс и пр.)
	11)	Установить размер изображения
	12)	Повернуть изображение
	13)	Кадрировать изображение

	14)	Создать слой
	15)	Дублировать слой
	16)	Удалить слой
	17)	Сгруппировать/разгруппировать слои

	18)	Открыть панель символов
	19)	Загрузить пакет шрифтов

	20)	Инструмент «Перемещение» для перемещения объектов
	21)	Инструмент «Прямоугольная область» для выделения области в форме прямоугольника

	23)	Инструмент «Выделение объектов» для поиска и автоматического выделения объектов
	24)	Инструмент «Рамка» для обрезки и расширения края изображения
	25)	Инструмент «Кадр» для создания кадров-местозаполнителей изображения
	26)	Инструмент «Пипетка» для взятия образца цвета с изображения
	27)	Инструмент «Точечная восстанавливающая кисть» для удаления отметок и пятен
	28)	Инструмент «Кисть» для рисования штрихов заказной кистью
	29)	Инструмент «Штамп» для рисования пикселями из другой области изображения
	30)	Инструмент «Архивная кисть» для восстановления предыдущего состояния фрагментов изображения
	31)	Инструмент «Ластик» для изменения цвета пикселей на фоновый или прозрачный
	32)	Инструмент «Градиент» для создания постепенного перехода между цветами
	33)	Инструмент «Размытие» для размывания области изображения
	34)	Инструмент «Осветлитель» для осветления области изображения
	35)	Инструмент «Перо» для создания и изменения контуров или фигур с опорными точками и маркерами
	36)	Инструмент «Горизонтальный текст» 
	37)	Инструмент «Выделение контура»
	38)	Инструмент «Фигура» для рисования заданной фигуры: прямоугольник, треугольник, многоугольник, линия, эллипс, произвольная фигура и пр.
	39)	Инструмент «Рука» для панорамирования по разным фрагментам изображения
	40)	Инструмент «Масштаб» для увеличения или уменьшения масштаба изображения
